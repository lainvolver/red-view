# 動作確認：コメント数推移機能および選択作品名の表示

## 実装内容
- ランキング種別の選択肢に「コメント数推移」を追加しました。
- グラフ表示機能を追加し、選択した単一の作品の各話ごとのコメント数を棒グラフで表示するようにしました。
- 凡例には、ランキング推移と全く同じ色・順序で全作品を表示させ、クリックすることでグラフに表示する作品を切り替えられる機能を追加しました。
- 初回表示時は最新話でコメント数が最も多い作品が自動で選択されるようにし、さらに他のランキング種別に切り替えても選択状態が維持されるように状態管理を追加しました。
- 「コメント数推移」グラフを表示している際、グラフの上部に現在選択されている作品名が大きく（`<h2>`タグを使用して）表示されるようにしました。
- ランキング推移など別のグラフ表示モードでは、この作品名は非表示になるように制御を追加しました。

## 確認事項
- [x] 「コメント数推移」を選択したとき、棒グラフが表示されること
- [x] 最初は最新話のコメント数が最も多い作品のグラフが表示されること
- [x] ランキング推移と同じ並び順・同じ色の凡例が右側に表示されること
- [x] 凡例の別の作品をクリックすると、その作品のコメント数推移グラフと上部の作品名表示が同時に切り替わること
- [x] ランキング種別を変更して再度「コメント数推移」に戻したとき、最後に選択していた作品のグラフが再度表示されること
- [x] グラフ上部に選択中の作品名が大きく表示されること
- [x] 他のランキング種別（「ランキング推移」など）を選んだ際は作品名が表示されないこと
